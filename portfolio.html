Portfólio

Aqui você encontra alguns dos meus projetos desenvolvidos durante meus estudos e experiências profissionais.

  • Projeto 1 – Este projeto simula um formulário de contato básico, permitindo que os usuários insiram nome, e-mail e uma mensagem. Ele demonstra o uso de form e input em HTML, essenciais para interações em páginas web.
  • Projeto 2 – Este exercício exibe uma tabela de produtos que agora inclui o cálculo automático do valor total de cada item. A soma é feita via JavaScript, utilizando querySelectorAll() para acessar os valores diretamente na tabela e atualizar a coluna correspondente.
  • Projeto 3 – Este projeto consiste em uma calculadora básica que realiza a soma de dois números. Ele demonstra a interação entre HTML, JavaScript e elementos dinâmicos da página, utilizando document.getElementById() para capturar valores e atualizar o conteúdo.
